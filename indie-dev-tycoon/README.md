# Indie Dev Pressure

Jogo web estatico em HTML, CSS e JavaScript.

## Estrutura

- `index.html`: tela principal e carregamento dos arquivos.
- `css/`: estilos visuais do jogo.
- `css/ui/`: estilos de telas especiais, como menu, selecao de personagem e transicoes.
- `assets/images/`: imagens da cena e personagens.
- `assets/audio/`: musicas e efeitos sonoros.
- `js/data/`: dados fixos, como plataformas, generos e reviewers.
- `js/systems/`: regras do jogo, simulacao, vendas, reviews, audio e save.
- `js/ui/`: renderizacao de telas, janelas e modais.
- `js/ui/main-menu.js`: comportamento do menu principal.
- `js/ui/character-selector.js`: escolha de personagem.
- `js/ui/transitions.js`: transicoes de entrada no jogo.
- `js/main.js`: inicializacao do jogo.

Para testar, abra `index.html` no navegador.
